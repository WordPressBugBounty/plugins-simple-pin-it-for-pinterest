Add a customizable Pinterest "Pin It" or "Save It" button to images in your posts.

== Description ==

Simple Pin It Button for Pinterest is a lightweight WordPress plugin that allows you to add a "Pin It" button overlay to images in your posts. The button is fully customizable through the plugin settings page, where you can adjust:

- Button location (Top Left, Top Right, Bottom Left, Bottom Right)
- Button background color
- Font color
- Enable or disable autohide
- Customize the "Pin It" text

Encourage your visitors to share your content on Pinterest with ease!

== Installation ==

1. Upload the `simple-pinterest-pin` folder to the `/wp-content/plugins/` directory, or install the plugin directly through the WordPress Plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Settings > Simple Pin It Button' to configure your options.

== Frequently Asked Questions ==

= Can I change the text on the button? =
Yes, you can customize the text (e.g., "Pin It", "Save It") from the plugin settings page.

= Does this plugin work with all themes? =
The plugin works with most WordPress themes.

= Is this plugin compatible with the latest WordPress version? =
Yes, Simple Pin It Button for Pinterest is tested with the latest WordPress version (6.7).
